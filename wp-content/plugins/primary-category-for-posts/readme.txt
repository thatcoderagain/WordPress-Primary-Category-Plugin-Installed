== Description ==
 The plugin provides ability to select a primary category for a post.
 The plugin also provides ability to query for posts (and custom post types) based on their primary categories.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/primary-category-for-posts` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Manage plugin setting from the Primary Category page under settings option.


== Frequently Asked Questions ==
Does it suppose the functionality to enable and disable primary category option for specific custom post types?
Yes it does support that. You can manage the plugin from Primary Category page settings under settings option.
